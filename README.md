# ESP32 UART to UDP Bridge

![Логотип](images/img0.png)

Проект превращает ESP32 в мост между UART и UDP. Устройство принимает данные по UART, отправляет их по UDP на указанный IP-адрес или в broadcast, а также показывает состояние на OLED-дисплее.

Сейчас проект поддерживает две аппаратные конфигурации:

- `ESP32-C3`
- `ESP32-S2 Mini`

## Возможности

- Передача данных из UART в UDP на порт `8888`
- Wi-Fi в режиме `STA` с автоматическим переходом в `AP`, если сеть недоступна
- Настройка параметров через веб-интерфейс `SettingsGyver`
- OLED-дисплей `SSD1306 128x64`
- Режим `broadcast`
- Режим `echo` для отладки
- Внешний экран по UDP на порт `82`
- Настройка мощности Wi-Fi
- Линия сброса внешнего устройства

## Поддерживаемые платы

### ESP32-C3

- Environment в `PlatformIO`: `esp32-c3-devkitm-1`
- Дисплей: `SSD1306 128x64` по `SPI`
- UART:
  - `TX = GPIO21`
  - `RX = GPIO20`
- Линия сброса внешнего устройства: `GPIO9`

Подключение дисплея:

```text
ESP32-C3     OLED SSD1306
GPIO 6   ->  MOSI
GPIO 4   ->  CLK
GPIO 3   ->  DC
GPIO 1   ->  CS
GPIO 2   ->  RESET
```

### ESP32-S2 Mini

- Environment в `PlatformIO`: `lolin_s2_mini`
- Дисплей: `SSD1306 128x64` по `I2C`
- Адрес дисплея: `0x3C`
- I2C:
  - `SDA = GPIO18`
  - `SCL = GPIO33`
- UART:
  - `TX = GPIO5`
  - `RX = GPIO3`
- При старте:
  - `GPIO35` устанавливается в `HIGH`
  - `GPIO37` устанавливается в `LOW`
- Линия сброса внешнего устройства: `GPIO37`

## Сборка

Установите `PlatformIO`, откройте проект и выберите нужное окружение.

Сборка для `ESP32-C3`:

```bash
platformio run -e esp32-c3-devkitm-1
```

Сборка для `ESP32-S2 Mini`:

```bash
platformio run -e lolin_s2_mini
```

Загрузка прошивки выполняется через то же окружение, которое вы собираете.

## Настройка

Через веб-интерфейс можно задать:

- IP клиента
- скорость UART
- режим `broadcast`
- режим `echo`
- параметры Wi-Fi
- использование внешнего экрана по UDP
- мощность Wi-Fi

## Сетевые параметры

- Передача данных по UDP: порт `8888`
- Внешний экран по UDP: порт `82`
- Скорость лога в `Serial`: `460800`

## Примечания

- На некоторых мини-платах Wi-Fi может работать нестабильно при мощности выше `8.5 dBm`. Если появляются проблемы, попробуйте уменьшить мощность через веб-интерфейс.
- Для `ESP32-S2 Mini` дисплей и UART уже разведены на разные пины, чтобы их можно было использовать одновременно.
- Для `ESP32-C3` исходная конфигурация проекта сохранена.
